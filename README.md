# Petition

#### npm start

Runs the app in the development mode.
Open http://localhost:8080 to view it in the browser.

The page will reload if you make edits.

## Technologies:

-   Handlebars
-   jQuery
-   canvas2D
-   Node.js
-   Express
-   Jest

## Features:

-   login and register
-   sign the petition and update the signature
-   update profile and password
